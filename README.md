## About The Project
<a href="https://github.com/MaxRuotsalainen">
    <img src="preview.png" alt="Logo" width="1100" height="450">
  </a>



This project is a web application built using Next.js, Bootstrap, and Prisma. The application provides a intuitive and responsive interface for users to create an account, log in, and edit their information as needed. 

The application is a CRUD (create, read, update, delete) application, which means that it allows users to create new records in the database, view and read existing records, update their information, and delete their account if desired. All user information is stored in a SQL database, ensuring that it is secure and easily accessible.

In addition to the core CRUD functionality, the application also provides features such as password reset and hashed passwords to ensure the security and authenticity of user accounts. With its sleek design and powerful features, this application makes it easy for users to manage and update their personal information.

## Getting Started

First, run the development server:

```bash
npm install
```
Then set up prisma and db and to run the server do..
```bash

npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
